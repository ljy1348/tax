▣ 수록요령

가. 입력화일형식
(1) 제출 가능 O.S명 : Windows 및 이와 호환되는 O.S
(2) 파일구조 : LINE SEQUENTIAL (각 레코드마다 CR/LF값 삽입)
(3) 전환방식 (CONVERSION MODE) : ASCII CODE
(5) 한글코드 : 2Byte완성형(KSC-5601 국가표준코드)

나. 파일명을 부여하는 방법
    일자 + 원천징수이행상황신고서 서식코드(C103900).신고구분상세코드
예)  20210207C103900.01






다. 유의사항
1. 한글로 표기되는 항목
- 한글항목에 한글 이외의 문자(한자, 영문자, 일본문자 등)을 표현하는 방법
  외국문자는 반드시 소리 나는 대로 한글로 표기합니다.
 (예)  “新實名”    →   “신실명” , “실명APT”  →   “실명아파트”

2. 성명
- 좌측부터 수록하여, 우측에 남는 자리는 빈칸(Space)으로 둡니다.
- “성명” 란에는 반드시 한 사람의 순수한 성명만을 수록하여야 합니다.
- 내국인의 성명
  성과 이름 사이에는 간격(공란)을 두지 말고 연이어서 수록합니다.
 (예)  홍길동		(○)	홍 길동		(X)
      남궁옥길	(○)	남궁 옥길	(X)
- 외국인의 성명
  소리 나는 대로 한글로 풀어서 수록합니다.
 (예)  JAMES MARTIN   →   “제임스마틴”

3. 주민등록번호(여권번호 등)
- 주민등록번호 사이에 “-“(하이픈 또는 마이너스)문자를 반드시 제거하여 숫자 13자리만 수록합니다.    (예)  123456-1234567   →   “1234561234567”

- 주민등록번호가 없는 자
	외국인등록번호, 여권번호, 재외국인록번호 등을 수록하며, 번호 사이의 ”-“ 또는 “ / ” 문자 등 특수문자는 반드시 제거하여 좌측부터 수록하며,  13문자가 안될 때는 우측에 남는 자리를 빈칸(Space)으로 둡니다.

- 주민등록번호의 구성체계는 다음과 같으며 작성시 오류가 없도록 유의하기 바랍니다.
        □ □ □ □ □ □  -  □  □  □  □  □  □  □
        └─┘└─┘└─┘   │  └─────┘ │ │
         년도   월    일     │  주민등록번호   │  검증코드
          └─────┘     │  부여기관번호   └  일련번호
             생년월일        │  (동·읍·면)
                           (성별)
                             └ 1900년대 출생자 → 남자 : 1. 여자 : 2.
                             └ 1800년대 출생자 → 남자 : 9. 여자 : 0.

4.	사업자등록번호(고유번호 ·납세번호)
- 사업자등록번호 사이에 “-“ (하이픈 또는 마이너스)문자를 반드시 제거하여 숫자 10자리만 수록합니다.
(예) 123-45-67890 → “1234567890”

5. 법인명(상호·조합명)
- 법인명, 조합명, 단체명, 상호 등은 좌측부터 수록하고 우측에 남는 자리는 공간(Space)으로둡니다.
- 영문자로 표기되는 법인명 등은 소리나는 대로 한글로 풀어서 수록합니다.

(예) SDS → “에스디에스”
	법인명 중 회사의 형태를 지칭하는 명칭은 약칭을 사용할 수 있습니다. 다만, 약칭을 사용하는 경우에는 반드시 약칭의 앞과 뒤에는 괄호를 사용하여야 합니다.
	
(예) 금융산업주식회사 (○)	금융산업 주	(X)
     금융산업㈜	  (○)	금융산업,주	(X)

(예) 조직형태별 약칭
주식회사(주)         합자회사(자)             합명회사(명)
유한회사(유)         사단법인(사)             재단법인(재)
학교법인(학)         복지법인(복)             의료법인(의)
종교법인(종)         기타법인(기) 
농업협동조합(농협)   수산업협동조합(수협)     축산업협동조합(축협)
임업협동조합(임협)   중소기업협동조합(기협)   상호신용금고(신금)
신용협동조합(신협)   새마을금고(새금)

6. 소재지(주소,거소)
- 소재지(주소,거소)는 반드시 한글로 수록합니다.
- 좌측부터 수록하고,우측에 남는 자리는 공란(Space)으로 둡니다
- (시.도),(구.시.군),(길),건물본번호,(동,아파트.상가) 순서로 수록합니다.
- 특별시,직할시,도 명은 축약하여 수록하거나 생략할 수 있습니다.
(예) “서울 특별시 영등포구 …..”    “ 서울 영등포구 …. ”	(○)
    “서울 특별시 영등포구 …..”    “ 영등포구…. ”		(○)
    “경상남도  …. ”		   “ 경남 …”			(○)
- 아파트(상가)명 또는 단지(동.호)명에 영문자로 표기된 부분이 있는 경우는 반드시 소리나는대로 한글로 풀어서 수록하여야 합니다.
(예) “…AID APT ….”        “….에이아이디아파트 ….”
    “…동서 BD 3F …”     “….동서빌딩3층 … .”

7. 전화번호
- 전화번호의 지역, 국, 번호 사이에는 “-“문자를 삽입하여야 합니다.
(예) 서울시 지역 647국 1234번  “02-647-1234”
- 좌측부터 수록하고, 남는 자리는 공란(Space)으로 둡니다.

8. 연월일 (제출연월일 등)
- 날짜항목에 연월일등을 수록할 경우에는 빈칸을 두지 말고 0(Zero)로 채워야 
합니다. 예) 2021년3월27일  “20210327” (월·일이 1단위일 경우 0을 앞에 수록)

9. 일련번호
- 일련번호는 순차적으로 부여합니다.
- 유효숫자는 우측을 기준으로 수록되어야 하며, 유효숫자 앞에는 반드시 빈칸(Space)을 두지 말고 0(Zero)로 채워야 합니다.
(예) 첫 번째 일련번호(6자리 수치로 구성)   “000001”
- 수치(0~9) 이외의 공란, 문자 등이 있으면 안됩니다.

10. 수  치
- 수치항목(계산에 사용되는 항목)의 유효숫자 앞에는 0(Zero)로 채워야 하며, 수치항목이 음수인 경우는 ‘-‘부호가 금액과 함께 표현되어야 합니다. 값이 없는 경우에도 0(Zero)로 채워야 합니다.

- 수치항목
      일련번호, 제출건수
      건수, 매수, 금액
      기타 4칙 연산 등에 사용되는 항목

- 수록형태별 설명
 수치 : 4칙 연산 등에 사용되는 항목으로 유효숫자 앞에는 반드시 0(Zero)로 채워
져야 하며, 공란(Space)이 있어서는 안됩니다.
(예) 세액이 5,230원인 경우  “0000005230”
 유효숫자 : 유효숫자는 우측을 기준으로 수록 되어야 하며 음수인 경우 ‘-‘ 을 표시합니다. 

(예)  양수 : 100, 음수 : -100  [단, 매입,매출계산서와 매입,매출세금계산서는 제외
                            (기존의 제출양식을 그대로 따름)]
      즉 길이가 6자리인 경우  음수 100은 -00100으로 수록합니다.
         길이가 6자리인 경우  양수 100은 000100으로 수록합니다.

 문자(숫자·영문자 포함) : 문자 또는 0~9까지의 숫자가 수록될 항목으로 좌로부터 수록하며, 우측에 남는 자리가 있는 경우에는 공란(Space)으로 둡니다.

▣ 제출, 징수(지급), 귀속연월 작성법
1. 정기신고서
(1) 매월신고
① (지급연월 = 제출연월 - 1) 이어야 합니다.
(기존에서는 지급연월이 제출연월과 같은 자료라 하더라도 신고가 가능하였으나
 제출연월이 2015년 3월분 이후에는 허용되지 않습니다. )

② 지급연월 >= 귀속연월 
( 징수(지급)연월은 급여, 퇴직금을 지급한 연월을 입력합니다. )

③ 소득처분 = ‘N’ 인 경우, 지급연월의 연도 = 귀속연월의 연도
※ 단, 소득처분 = ‘Y’ 인 경우이거나 또는 근로/퇴직소득의 지급의제로 인한 경우는
지급연월의 연도 >= 귀속연월의 연도로 입력 가능합니다.
(근로/퇴직소득의 지급의제의 예시는 아래에서 설명)
(2) 반기신고
① 계속사업자가 신고하는 경우, 귀속연월은 반기 시작월, 지급연월은 반기 종료월로 
신고해야 합니다.
상반기 제출분의 경우 귀속연월은 1월, 지급연월은 6월, 제출연월은 7월
하반기 제출분의 경우 귀속연월은 7월 지급연월은 12월, 제출연월은 다음연도 1월
